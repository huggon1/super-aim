# Six Target Ultimate

V1 is a 60 second 3D FPS training mode.

- Six targets are active at all times.
- Targets spawn in front of the player within a configurable horizontal angle, distance range, and height range.
- A hit immediately respawns the same target at a new valid position.
- Valid positions avoid placing targets too close to other active targets.
- The mode ends when the timer reaches zero and sends results to the result screen.

V1 success means the user can start from the menu, play a full session, and see final score, hits, shots, and accuracy.
